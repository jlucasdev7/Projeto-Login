## João Lucas | Projeto Login

##

O projeto de Login foi desenvolvido com o objetivo de aperfeiçoar meus conhecimentos
em HTML e CSS, criando uma interface moderna e funcional para autenticação de usuários.

##

Durante o processo, utilizei HTML semântico para estruturar o formulário de login de forma 
clara e acessível, e CSS3 para estilizar os elementos com foco em design limpo, alinhamento 
preciso e boa usabilidade.

##

Também implementei responsividade utilizando media queries, garantindo que o layout se adaptasse 
corretamente a diferentes dispositivos — desde smartphones até telas maiores.

##

Esse projeto me permitiu aprimorar habilidades em posicionamento de elementos, boas práticas de 
estilização, design responsivo e atenção à experiência do usuário, consolidando minha base no 
desenvolvimento front-end.
